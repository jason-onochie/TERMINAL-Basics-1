To Create A Directory

To create a directory in windows terminal, run command prompt and type mkdir name-of-directory then press ENTER

To Create A File

To create a file in a directory in windows terminal, run command prompt, locate the directory in which the file is to be created by typing cd path-to-directory and press the enter button.
When in the directory, type in the following : type null > name-of-file.file-type then press ENTER

To List Contents Of A Directory

To list contents of a directory in windows terminal, locate the directory by typing cd path-to-directory and press the enter button.
When in the directory, type dir then press ENTER. This will list all sub-directories and files in the parent directory.
